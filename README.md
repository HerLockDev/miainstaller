# Mia Installer

### MiaUserbot Otomatik Deployer

[Mia UserBot Reposu](https://github.com/MiaUserBot/mia)

## Kurulum
```sh
git clone https://github.com/MiaUserBot/mia
cd installer
pip install -r requirements.txt
python3 -m mia_installer
```
